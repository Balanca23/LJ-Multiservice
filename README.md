# 🧼 Serviços de Limpeza Profissional

Este é um website criado para apresentar serviços profissionais de limpeza, incluindo limpeza residencial, empresarial, pós-obra e manutenção de espaços.

## 🧹 Sobre o projeto

Este site foi desenvolvido com o objetivo de divulgar serviços de limpeza profissional, facilitando o contato com clientes e a apresentação dos serviços oferecidos.

O foco é oferecer uma apresentação simples, moderna e acessível.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
